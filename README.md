### ✧ Installation

To install **Lune** via Scoop, run the following commands in your terminal:

```powershell
# 1. Add this bucket
scoop bucket add saraansx https://github.com/saraansx/scoop-bucket

# 2. Install Lune
scoop install lune
```

<br />

### ✧ Updating

Lune is configured with **Automated Version Tracking**. When a new release is published in the main repository, this manifest is updated instantly. 

To update your local installation:
```powershell
scoop update lune
```

<br />

### ✧ Uninstallation

```powershell
scoop uninstall lune
```
